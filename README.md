# TV 3D Launcher

  好久以前做的一个小应用，公司项目没用上，所以拿出来分享一下找点存在感。

  针对TV电视开发的 3D画廊 桌面小应用 。
   
  仅仅支持电视机和tv模拟器安装，手机上安装界面会错乱。

  需要用到遥控器操作，不支持触控。


  功能列表：
  
      1. 3d 画廊

      2. 本地应用管理列表

      3. 应用置顶排序

      4. 应用静默卸载（系统权限，需要系统签名使用）
